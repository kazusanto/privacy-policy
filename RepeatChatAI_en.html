Privacy Policy

Information We Collect

This app handles only the user-supplied OpenAI API Key and the chat messages you send (“User Content”).
No other personal data such as your name, email address, or location is collected.

Purpose of Use

The API Key is used solely as an authentication token for accessing the OpenAI API and for no other purpose.

Storage and Deletion

The API Key is stored in an encrypted state within the device’s Keychain. Keychain is Apple’s system-level secure storage, protected by the user’s authentication, and is inaccessible to other apps or third parties. To delete the API Key, tap “Delete API Key” in the settings screen. Note that uninstalling the app may leave the Keychain item intact; if so, reinstall the app and press the same button to remove it completely.

Third-Party Sharing

The API Key is transmitted only to OpenAI when communicating with the OpenAI API and is not shared with any third party, including the developer.

Data Collected by Third Parties

OpenAI may retain request and response logs—such as prompts, responses, and IP addresses—for the purposes of abuse monitoring and service quality. For retention periods and detailed handling, please refer to the OpenAI Privacy Policy.
