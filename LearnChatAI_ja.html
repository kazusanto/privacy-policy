プライバシーポリシー

収集する情報

本アプリはユーザーが入力する OpenAI API Key とチャットメッセージ （以下「ユーザーコンテンツ」）のみを取り扱います。
氏名、メールアドレス、位置情報など、それ以外の個人データは収集しません。

利用目的

API Key は OpenAI API への認証トークンとしてのみ使用し、ユーザーコンテンツは AI 応答を生成する目的でのみ OpenAI に送信します。

保管と削除

API Key は端末内の Keychain に暗号化された状態で保存されます。 Keychain はユーザー認証で保護された Apple のシステムストレージであり、他アプリや 第三者からはアクセスできません。
API Key を削除したい場合は、設定画面の「API Key を削除」ボタンをご利用ください。 アプリをアンインストールしても Keychain に残る場合がありますが、再インストール後に 同ボタンを押すことで完全に削除できます。

第三者によるデータ保持

OpenAI 社はリクエストログ（プロンプト、レスポンス、IP アドレスなど）を 不正利用の防止やサービス品質向上の目的で保持する場合があります。
保持期間や具体的な取扱いについては、 OpenAI プライバシーポリシー をご確認ください。
