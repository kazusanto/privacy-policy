プライバシーポリシー

収集する情報

本アプリはユーザーが保有する OpenAI API Key のみを取り扱います。氏名、メールアドレス、位置情報等の個人データは一切収集しません。

利用目的

取得した API Key は OpenAI API への認証トークンとしてのみ使用し、他の目的では利用しません。

保管と削除

API Key は端末内の Keychain に暗号化された状態で保存されます。Keychain はユーザー認証で保護された Apple のシステムストレージであり、他アプリや第三者からはアクセスできません。API Key を削除したい場合は、設定画面の「API Key を削除」ボタンをご利用ください。アプリをアンインストールしても Keychain に残る場合がありますが、アプリを再インストール後に同ボタンを押すことで完全に削除できます。

第三者提供について

API Key は OpenAI との通信時にのみ送信され、開発者を含む第三者に共有されることはありません。
